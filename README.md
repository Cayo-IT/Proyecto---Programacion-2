## Proyecto---Programacion-2
Aqui se vera los avances y discusion del proyecto final sobre nuestro juego del tipo bullet hell 
------------

# De que se trata esto

Es un proyecto en el cual se trata de crear un videojuego en el entorno de Python del genero 
bullet hell. 
La idea es que se trate de un personaje que esta encima de una nave y que este se encarge de 
derrotar enemigos y llegar hasta el final.
-----------
# A considerar

El juego va a funcionar mediante un bucle que se le conoce como "Game Loop"
Se tiene pensado usar imagenes licenciadas 
---------
# Elementos Clave

- Scrolling infinito del fondo (La sensacion de vuelo)
- Los Sprites
- Patrones de movimiento
- Musica, Sonidos, Dialogos?
- QA (Control de Calidad)
------
# Roles
La manera a trabajar puede ser la siguiente:

Hay 3 secciones vitales para la creacion de este juego

- El sistema (Estructura del entorno, el Game Loop, gestión de niveles y estados (menú, juego, Game Over))
- Gameplay/Mecanicas (Movimiento del jugador, tipos de disparos, patrones de enemigos y colisiones)
- Arte, Sonido y Nivel (Creación de sprites (naves, balas), fondos, efectos de sonido y diseño de oleadas de enemigos.
